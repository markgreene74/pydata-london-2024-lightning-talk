## Setup

- install the requirements (`jupyterlab`, `seaborn`, `jupytext`, `pre-commit`)
    ```shell
    pip install -r requirements.txt
    ```
- install the `pre-commit` hooks
    ```shell
    pre-commit install
    ```
- start Jupyter
    ```shell
    jupyter-lab
    ```
